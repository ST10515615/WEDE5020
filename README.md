# WEDE5020
Web Development project
## Project Title
Shoprite
## Proposal Content
1. Organisation Overview 
Shoprite Holdings

“Shoprite Holdings is the largest supermarket retailer in Africa and has its origins in South Africa. The company was founded in 1979 when a small group of investors purchased eight grocery stores in the Western Cape.” (Shoprite Holdings, 2025)

Mission 
“Our mission is to deliver low prices in a world-class shopping environment to customers across the African continent.” (Shoprite Holdings, 2025) 

Vision 

“To be Africa’s most profitable omnichannel retailer.” (Shoprite Holdings, 2025) 

Target market 

Shoprite’s target audience is for lower to middle-income consumers. 

 

2. Website Goals and Objectives 

The main goal of the website is to help the company sell its products in its stores by using online shopping. 

The website wants to make selling products easy and user-friendly. 

The website needs to work well on every device.  

 

3. Current Website Analysis 

Strengths 

The Shoprite brand is internationally known. 

It shows product and specials information. 

The website shows locations of Shoprite branches. 

Weakness 

The website is hard to use. 

The website is also hard to use for online shopping. 

The website is not user-friendly on a mobile device 

 

 Areas to improvement 

I’ll put a search bar on the website. 

I’ll put a way to buy things on the website. 

I’ll make the website work on every device. 

 

4. Proposed Website Features and Functionality 

The Home page will have things like current promotions, products on special, and important announcements. 

The About Us Page will include the history of Shoprite, its mission, vision, and values. 

The Product/Services Page will show products categorised. And there will be a search bar. 

The Online Shopping functionality will help customers to make an account and buy products online. 

The Contact Page will have contact information like phone numbers, email addresses, and store locations. 

 

5. Design and User Experience 

Colour Scheme 

 red, white, and yellow. 

Typography 

Headings will be bold to highlight categories and promotions. 

Body text should be simple 

Font size will be big enough for mobile devices. 

Layout and Design 

The home page will show products, specials, and categories. 

Product pages should have images, a price, and a description of the products. 

User Experience 

Website will be easy to use 

The website will have a Home, Product, About, and Contact. 

The layout will work well on every device 

 

6. Technical Requirements 

Hosting Requirements 

The website will use cloud hosting to handle traffic and provide good performance. Furthermore, the hosting platform should provide fast loading and strong security features. 

Domain Name 

The domain name should be www.shoprite.co.za because it is professional, easy to remember, and reflects the brand's name. 

 

Programming language 

The website will use HTML, CSS, and JavaScript. 

Data Requirements 

The website will use MySQL to store product information, customer data, and transaction records. 

Security Requirements 

For protection from cyber threats. The website will use SSL encryption, a secure payment gateway, and a firewall.
